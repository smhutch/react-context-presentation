## Modal display logic
```jsx
{showModal && <Modal />}
```

<Appear>
  <p><S>Props</S> ‚Äî Write every time.</p>
  <p><S>Context</S> ‚Äî Write once.</p>
</Appear>

---

## Modal state
```jsx
this.state = {
  modal: null
}
```

<Appear>
  <p><S>Props</S> ‚Äî Write every time. Potential for multiple modals.</p>
  <p><S>Context</S> ‚Äî Write once. Only one modal can be shown at a time.</p>
</Appear>

---

## Opening the Modal
```jsx
<Button onClick={() => openModal()} />
```

<Appear>
  <p><S>Props</S> ‚Äî May need to pass through many components.</p>
  <p><S>Context</S> ‚Äî Import the context wherever you need it.</p>
</Appear>
